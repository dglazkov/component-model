.1 Overview

By default, the component should not be able to affect the including document or parent component, nor be able to discern its state (confinement), and vice-versa (isolation).

Confinement and isolation can be broken if and only if both the including document/parent component and the component "agree".

.1 JavaScript

Confinement requires:

  • The component has no access to document or window, or the returned objects are limited.
  • CSS rules do not cross over into the shadow tree, nor our of it.
  • ...

Isolation requires:

  • There is no accessor method to reach the shadow tree from the enclosing scope.
  • CSS rules do not cross over into the shadow tree, nor our of it.
  • ...
